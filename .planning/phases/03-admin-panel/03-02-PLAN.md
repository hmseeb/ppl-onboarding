Create `components/admin/CopyLinkButton.tsx`:
- "use client" component.
- Props: `{ token: string }`.
- Constructs the URL: `${window.location.origin}/onboard/${token}`.
- On click: `navigator.clipboard.writeText(url)`.
- Shows a copy icon (lucide-react `Copy`) that changes to a check icon (`Check`) for 2 seconds after copying.
- Use useState for the copied state, setTimeout to reset.
- Use shadcn Button with `variant="ghost"` and `size="sm"`.

Create `components/admin/BrokerTable.tsx`:
- "use client" component (needs clipboard interaction via CopyLinkButton).
- Props: `{ brokers: Broker[] }` (import Broker from `@/lib/types`).
- Uses shadcn Table, TableHeader, TableBody, TableRow, TableHead, TableCell.
- Columns:
  1. **Name** — `${broker.first_name} ${broker.last_name}`
  2. **Company** — `broker.company_name` or dash if null
  3. **Status** — Render as a shadcn Badge component:
     - `not_started`: variant="secondary", text "Not Started"
     - `in_progress`: variant="outline" with yellow/amber text, text "In Progress"
     - `completed`: variant="default" (red primary), text "Completed"
  4. **Received** — `broker.created_at` formatted as short date (e.g., "Mar 11, 2026") using `new Date(broker.created_at).toLocaleDateString('en-US', { month: 'short', day: 'numeric', year: 'numeric' })`
  5. **Completed** — `broker.completed_at` formatted same way, or dash if null
  6. **Delivery** — `broker.delivery_method` displayed as uppercase badge or dash if null
  7. **Link** — `<CopyLinkButton token={broker.token} />`

- If brokers array is empty, show a centered message: "No brokers yet. Waiting for the first GHL webhook."
- Table should be responsive: wrap in a `div` with `overflow-x-auto` for mobile.
- Match dark theme styling — table borders use border-border color, header row slightly lighter (bg-muted).
