1. Create `components/onboarding/OnboardingStepper.tsx` — Client Component state machine:
   - "use client"
   - Import `useState, useCallback, useEffect` from react
   - Import `Broker` from `@/lib/types`
   - Import `OnboardingHeader` from `./OnboardingHeader`
   - Import `StepTransition` from `./StepTransition`
   - Import Step1Welcome and Step2Delivery (steps 3-7 will be added in later plans)
   - Props: `{ broker: Broker; token: string }`
   - State:
     - `currentStep` initialized from `broker.current_step ?? 1`
     - `formData` as `Record<string, unknown>` initialized to `{}`
   - `useEffect` on mount — fire POST to `/api/brokers/${token}/start` if `broker.status === 'not_started'` (ONBD-12). Use an AbortController for cleanup. Only fire once.
   - `goToStep(step: number)` — sets currentStep, calls `PATCH /api/brokers/${token}/step` with `{ step }` in the body (ONBD-13). Fire-and-forget (don't block UI).
   - `handleNext(stepData?: Record<string, unknown>)` — merge stepData into formData, call `goToStep(currentStep + 1)`. Cap at 7.
   - `handleBack()` — call `goToStep(currentStep - 1)`. Floor at 1.
   - `handleComplete(stepData?: Record<string, unknown>)` — merge stepData, POST to `/api/brokers/${token}/complete` with accumulated formData. On success, redirect to `/onboard/${token}/complete` using `window.location.href`.
   - Render:
     ```
     <div className="min-h-screen bg-background text-foreground">
       <OnboardingHeader currentStep={currentStep} totalSteps={7} />
       <main className="max-w-lg mx-auto px-4 pb-24">
         <StepTransition stepKey={currentStep}>
           {currentStep === 1 && <Step1Welcome broker={broker} onNext={handleNext} />}
           {currentStep === 2 && <Step2Delivery broker={broker} onNext={handleNext} onBack={handleBack} />}
           {/* Steps 3-7: placeholder for now - will be added in Plans 03 and 04 */}
           {currentStep >= 3 && currentStep <= 7 && (
             <div className="py-12 text-center text-muted-foreground">
               Step {currentStep} — coming soon
               <div className="mt-4 flex gap-2 justify-center">
                 <button onClick={handleBack}>Back</button>
                 {currentStep < 7 ? (
                   <button onClick={() => handleNext()}>Next</button>
                 ) : (
                   <button onClick={() => handleComplete()}>Complete</button>
                 )}
               </div>
             </div>
           )}
         </StepTransition>
       </main>
     </div>
     ```
   - IMPORTANT: The placeholder for steps 3-7 is temporary — Plans 02-03 and 02-04 will replace them with real step components.
